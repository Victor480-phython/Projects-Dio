# Projeto Empresarial: Pipeline ETL + Análise de Vendas

## Descrição
Projeto profissional desenvolvido em Python puro e gratuitos, simulando pipeline de ETL real de uma empresa de e-commerce.
O objetivo é deixar claro que o pipeline ETL é o coração de um Analista de Dados: extrair dados brutos, transformá-los e carregá-los para análises e dashboards.

Execute sempre a partir de `src/main.py`.

## Estrutura (Flat)
Existem as pastas padrão, mas toda a lógica reside em `src/` (módulos simples para começar) e dados brutos em `database/`.
