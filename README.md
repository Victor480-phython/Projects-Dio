## 🎯 Objetivo do Projeto

Os relatórios do **Power BI** foram desenvolvidos com o objetivo de criar uma **interface refinada e de entendimento simples** para a **tomada de decisões gerenciais**.  

Para garantir um design harmonioso, equilibrado e profissional, apliquei a **proporção áurea** no layout dos relatórios financeiros. Essa técnica de design proporciona uma experiência visual mais agradável e intuitiva.

Além disso, criei **parâmetros e métricas na linguagem DAX** (Data Analysis Expressions) para auxiliar nos cálculos estatísticos, permitindo análises dinâmicas, segmentações e insights

## 🛠️ Tecnologias Utilizadas

- **Excel** – Dashboards e análises tabulares
- **Power BI** – Modelagem de dados, visualizações interativas e relatórios
- **Linguagem DAX** – Criação de medidas, colunas calculadas e parâmetros
- **Proporção Áurea** – Aplicada no design dos relatórios financeiros
- **My SQL Workbench** – Uma das bases utilizadas para demonstrações genéricas

## 📁 Estrutura do Repositório

Cada arquivo está separado com uma descrição clara para facilitar a navegação e o entendimento.

## 📊 Dashboard_Vendas_Xbox.xlsx
Tipo: Microsoft Excel
Dashboard interativo de análise de vendas do produto Xbox. Contém gráficos dinâmicos, tabelas dinâmicas e KPIs de desempenho para monitoramento rápido das vendas.
## 📈 Financial Sample.pbix
Tipo: Power BI
Relatório base com o modelo de dados financeiro de amostra. Serve como foundation para os demais relatórios, contendo as tabelas principais e relacionamentos iniciais.
## ⚙️ PBI_DAX.pbix
Tipo: Power BI
Projeto focado exclusivamente em DAX (Data Analysis Expressions). Demonstração prática de medidas, parâmetros e cálculos estatísticos avançados utilizados nos relatórios.
## 📋 Relatorio Completo com tudo.pbix
Tipo: Power BI
Relatório completo que integra todos os dados, visualizações, segmentadores e métricas DAX. Oferece uma visão 360° do negócio, ideal para análises abrangentes.
## 🎯 Relatorio para decisões gerenciais parametros.pbix
Tipo: Power BI
Relatório otimizado para tomada de decisão gerencial. Utiliza parâmetros interativos para simulações de cenários e análises dinâmicas, facilitando o suporte estratégico.
## 🔍 Relatório com Segmentadores Financial Sample.pbix
Tipo: Power BI
Relatório que utiliza segmentadores (slicers) avançados aplicados ao modelo Financial Sample. Perfeito para filtragem rápida e visualização segmentada dos dados

> 📌 **Como usar:** Basta baixar o arquivo `.pbix` e abrir diretamente no **Power BI Desktop**. O arquivo `.xlsx` abre normalmente no Excel

## 🔧 Como os Relatórios Foram Desenvolvidos

1. **Modelagem de Dados** – Relacionamentos entre tabelas no modelo de dados do Power BI.
2. **Medidas DAX** – Criação de métricas personalizadas (ex: Total Vendas, Margem de Lucro %, Crescimento %, etc.)
3. **Parâmetros** – Utilizados para filtros dinâmicos e simulações de cenários.
4. **Design com Proporção Áurea** – Espaçamento, alinhamento e proporção dos visuais seguem a regra de ouro para maior harmonia visual
5. **Interatividade** – Segmentadores, botões de navegação e tooltips personalizados para facilitar a experiência do usuário final (gestores)
